# Base Habit Orbit

## App Name

Base Habit Orbit

## Tagline

Keep one habit in orbit

## Description

Log one habit completion on Base and keep a clean orbit record with streak count and lookup by ID.

## Category

Habit / streak / onchain utility

## Short Pitch

Base Habit Orbit gives people a simple way to log a habit completion on Base and keep a clean orbit record with a running count.

## Production Environment Variables

- `NEXT_PUBLIC_HABIT_ORBIT_CONTRACT_ADDRESS`: deployed habit orbit contract address
- `NEXT_PUBLIC_BUILDER_CODE`: Base Builder Code after the app is registered
- `NEXT_PUBLIC_BASE_APP_ID`: Base.dev app id for domain ownership verification
